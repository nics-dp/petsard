## Storage Location

Benchmark datasets are stored in AWS S3:
- **Organization**: NICS-RA (National Institute of Cyber Security)
- **Bucket**: `petsard-benchmark`

## Available Datasets

The system automatically reads the list of available datasets from the `petsard/loader/benchmark_datasets.yaml` configuration file.
